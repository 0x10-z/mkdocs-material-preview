# Complex Admonition Test

This file tests admonitions with rich nested content.

!!! Warning "Attention!"

    If any part contours overlap, all affected parts are colored red.

!!! Info

    The dimensions are shown in the cutting plan's system of units.

- Item two

??? "![](https://placehold.co/16) Undo"

    To undo the last operation performed.

??? "![](https://placehold.co/16) Restore"

    Restores the last operation undone.

??? "![](https://placehold.co/16) Zooming functions"

    You can zoom cutting plans by hand or by pressing the appropriate button.

    * Zooming in and out by hand
    * Fits the cutting plan to the window

??? "Select parts automatically"

    This group contains all functions for automatic selection of parts.

    * Selects all parts of the cutting plan.
    * Selects all instances of the selected parts.
    * Inverts the selection of parts.
    * Cancels the selection of parts.

??? "![](https://placehold.co/16) Select parts with frame"

    Selects parts by dragging a frame around the parts to be selected.

    !!! Info

        Only parts that are completely inside the frame are selected.

??? "![](https://placehold.co/16) Move part"

    Moves the selected part.

    !!! Info

        Tap the icon to switch to Move mode.

??? "![](https://placehold.co/16) Copy part"

    Moves a copy of the selected part.

    !!! Info

        Tap the icon to switch to Copy mode.

??? "![](https://placehold.co/16) ![](https://placehold.co/16) Select multiple-single parts manually"

    Defines the mode of selecting parts.

    * Only the selected part is highlighted.
    * All selected parts are highlighted.

    !!! Tip

        Tap on a selected part to deselect it.

??? "![](https://placehold.co/16) ![](https://placehold.co/16) Rotate part"

    * Rotates the part in steps defined in the settings (default of 45°) to the left.
    * Rotates the part in steps defined in the settings (default of 45°) to the right.

??? "![](https://placehold.co/16) Nest part in rows"

    Nests a part or a selection of parts in rows and columns.

??? "![](https://placehold.co/16) Mirror part"

    Mirrors the selected part around the X axis.

??? "![](https://placehold.co/16) Delete part"

    1. Select part.
    2. Drag the selected part onto the icon at the edge of the screen and drop it there.

    !!! Tip

        You can also delete multiple selected parts.

??? "![](https://placehold.co/16) Save part in parts pool"

    1. Select part.
    2. Drag the selected part onto the icon at the edge of the screen and drop it there.

    !!! Tip

        You can also copy multiple selected parts to the parts pool.

??? "![](https://placehold.co/16) Shorten residual grid cut"

    Shortens the residual grid cut by dragging out a box to the next intersection point.

    !!! Info

        The icon is only displayed if the Residual grid cuts function was selected.
